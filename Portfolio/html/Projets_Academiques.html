Projet classification (automatisé) Java

Les compétences mises en œuvre

  • Implémenter des conceptions simples
  • Élaborer des conceptions simples
  • Analyser un problème avec méthode (découpage en éléments algorithmiques simples, structure de données…)
  • Comparer des algorithmes pour des problèmes classiques (tris simples, recherche…)
  • Formaliser et mettre en œuvre des outils mathématiques pour l’informatique
  • Faire des essais et évaluer leurs résultats en regard des spécifications
  • Nombre de personnes sur le projet : 2
  • Durée : 1 semaine
  • Objectifs : créer différents programmes en Java permettant de classer des dépêches et de tester leur efficacité
  • Rendu : Le programme Java et un compte rendu en anglais
  • Contenu du projet :

    • Nous avons mis en place trois programmes de classification de dépêches, le premier en créant un dictionnaire manuel, un second en faisant un programme permettant de créer le dictionnaire, et une dernière méthode, simplifiée pour le projet, plus utilisée de nos jours pour classer des dépêches (le KNN). En parallèle, nous devions écrire un compte rendu en anglais. Pour cela, j’ai réfléchi à la structure des différents codes et j’ai codé les parties plutôt simples, laissant à mon coéquipier les différentes parties lui permettant de coder le reste. Je me suis aussi occupé du compte rendu en anglais.

Projet d'installation d'un poste de développement Linux

Les compétences mises en œuvre

  • Identifier les différents composants (matériels et logiciels) d’un système numérique
  • Utiliser les fonctionnalités de base d’un système multitâches / multiutilisateurs
  • Installer et configurer un système d’exploitation et des outils de développement
  • Configurer un poste de travail dans un réseau d’entreprise
  • Formaliser et mettre en œuvre des outils mathématiques pour l’informatique
  • Nombre de personnes sur le projet : 1
  • Durée : 1 demi-journée
  • Langages abordés : HTML
  • Objectifs : Installation d’un poste pour le développement sous IntelliJ
  • Contenu du projet :

    • Pour cela, j’ai commencé par installer Debian sur la station, puis j’ai paramétré la machine virtuelle. Une fois cela fait, j’ai pu lancer la station et finir de la paramétrer de l’intérieur. Enfin, vu que le projet devait permettre l’utilisation par une personne faisant du développement, j’ai installé les packages nécessaires. En parallèle du processus d’installation, nous devions produire une carte mentale permettant l’installation d’un poste de travail pour le développement.

Concevoir une base de données

Les compétences mises en œuvre

  • Mettre à jour et interroger une base de données relationnelle (en requêtes directes ou à travers une application)
  • Visualiser des données
  • Concevoir une base de données relationnelle à partir d’un cahier des charges
  • Nombre de personnes sur le projet : 2
  • Durée : 1 mois
  • Objectifs : Concevoir une base de données utilisable en partant de données open source
  • Rendu : Règle de gestion + Schéma Entité Association + Schéma Logique Relationnel (SLR) + fichier de création des tables + fichier de manipulation des tables
  • Contenu du projet :

    • Pendant ce projet, nous avons développé nos compétences de manipulation d'une base de données en commençant par le repérage des différentes tâches que nous allions réaliser. Ensuite, les différents schémas nous ont permis de modéliser nos premières idées. Enfin, les tests à réaliser lors de la dernière étape nous ont montré les failles de nos précédents schémas et nous ont permis d'adapter notre base de données.

Projet création de site web (Hardis groupe)

Les compétences mises en œuvre

  • Appréhender les besoins du client et de l’utilisateur
  • Mettre en place les outils de gestion de projet (Whimsical)
  • Conduire un projet
  • Identifier les statuts, les fonctions et les rôles de chaque membre d’une équipe pluridisciplinaire
  • Acquérir les compétences interpersonnelles pour travailler en équipe
  • Nombre de personnes sur le projet : 3
  • Durée : 3 semaines (1 semaine recueil des besoins + 1 semaine grille de projet et préparation orale + 1 semaine code du site)
  • Langages abordés : HTML, CSS (+ un peu de JavaScript)
  • Objectifs : Création d'un site web pour une entreprise du bassin grenoblois
  • Contenu du projet :

    • Nous avons commencé par le recueil des besoins de l'entreprise en regardant leur site actuel. C'était notre premier projet de l'année donc mise en place de chacun, j'ai pris la tête du projet pour la fin de cette première étape et le reste du projet. Nous avons ensuite rempli notre grille nous permettant de voir quels étaient les aspects sur lesquels nous avions besoin de progresser. Mais aussi ce dont nous avions besoin pour rendre notre site le plus proche possible de ce que l'entreprise voulait (la consigne étant de leur refaire un site web facilitant l'insertion des élèves de troisième pour leur stage). Enfin, nous avons codé le site en utilisant les outils que nous avions vus en cours, notamment Visual Studio Code pour la partie HTML, CSS, et JavaScript.
