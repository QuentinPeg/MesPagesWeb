Projet création de site web (Hardis groupe)

    Les compétence mise en oeuvre

  • Appréhender les besoins du client et de l’utilisateur
  • Mettre en place les outils de gestion de projet(Whimsical)
  • Conduire un projet
  • Identifier les statuts, les fonctions et les rôles de chaque membre d’une équipe pluridisciplinaire
  • Acquérir les compétences interpersonnelles pour travailler en équipe
  • Nombre de personne sur le projet : 3
  • Durée : 3 semaine (1 semaine recueil des besoins + 1 semaine grille de projet et préparation oral + 1 semaine code du site)
  • language abordée : HTML, CSS (+ un peu de JavaScript)
  • Objectifs : Création d'un site web pour une entreprise du bassin grenoblois
  • Contenu du projet :

    • Nous avons commencé par le receuil des besoins de l'entreprise en regardant leur site actuel. C'était notre premier projet de l'année donc mise en place de chacun, j'ai pris la tête du projet pour la fin de cette première étape et le reste du projet. Nous avons ensuite rempli notre grille nous permmettant de voir quels était les choses sur lesquelles nous avions besoins de progresser. Mais aussi ce que nous allions avoir besoins pour rendre notre site le plus proche possible de ce que l'entreprise voulait (la onsigne étant de leur refaire un site web facilitant l'insertion des élèves de troisième pour leur stage). Enfin nous avons codé le site en utilisant les outils que nous avions vu en cours, notamment VisualStudio Code pour la partie Html, Css et javascript.
